# <img src="../icon.png" alt="drawing" width="26"/> Cognitive Lens


**Cognitive Lens** helps you identify complex methods in your TypeScript and C# code.

### Features
* **CodeLens**: Displays complexity score above methods.
* **Inlay Hints**: Shows complexity contribution of code structures.
* **Gutter Icons**: Visual indicators for complexity levels.
* **Diagnostics**: Highlights methods exceeding complexity thresholds.
