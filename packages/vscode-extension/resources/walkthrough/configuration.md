# Configuration

Customize **Cognitive Lens** to fit your workflow.

### Thresholds
Adjust the complexity limits:
* **Warning**: Default 15.
* **Error**: Default 25.

### Visual Feedback
Toggle specific features:
* **CodeLens**: Show score above methods.
* **Gutter Icon**: Show icon in the margin.
* **Inlay Hints**: Show detailed complexity breakdown.

[Configure Settings](command:workbench.action.openSettings?%22cognitiveComplexity%22)
